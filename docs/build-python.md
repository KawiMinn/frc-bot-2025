# Robotpy
Building and deploying python

## Install Python
```
sudo apt install python3 python3-pip
```

For mac and PC, use whatever means is most reliable for you.

## Install Robotpy
```
pip3 install robotpy
```

## Sync up
```
robotpy sync
```

## Simulate
```
robotpy sim
```
